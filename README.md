# Blockchain-Based Fake Product Detection

## Description

This project focuses on detecting counterfeit products using a blockchain-based authentication system with QR verification. It ensures transparency, traceability, and secure validation across the supply chain.

## Features

* QR-based product verification
* Blockchain-based authentication
* Tamper-proof data validation
* Real-time product verification

## Technologies

* Blockchain (concept)
* Web Application
* QR Code System
